# Requirements

- Install [NodeJS](https://nodejs.org/en/download/)
- Install [MySQL](https://www.mysql.com/downloads/)


# Development

```
- Run `db.sql` file in folder `configs`
```

# Deployment

```
$ npm install
```
```
$ npm start
```
